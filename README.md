Titanic Survival Prediction using Machine Learning
📌 Project Overview

This project focuses on predicting whether a passenger survived the Titanic disaster using machine learning techniques. The model is built using the famous Titanic dataset and demonstrates the complete ML workflow from data preprocessing to model evaluation.

📊 Dataset

The dataset used in this project contains information about passengers such as:

Age
Gender
Passenger Class (Pclass)
Fare
Embarked Port
Survival Status

⚙️ Technologies Used
Python 
Pandas & NumPy
Matplotlib & Seaborn (for visualization)
Scikit-learn

🔍 Project Workflow
Data Loading – Imported the Titanic dataset
Data Cleaning – Handled missing values and removed unnecessary columns
Feature Engineering – Converted categorical data using encoding
Train-Test Split – Split data into training and testing sets

Model Building – Applied:
Logistic Regression
Decision Tree
Random Forest

🤖 Machine Learning Models Used
Logistic Regression
Decision Tree Classifier
Random Forest Classifier

📈 Results
The models were trained and tested on the dataset, and performance was evaluated based on accuracy. Among the models, Random Forest provided better performance compared to others.
